## 📌 About

This repository contains hands-on programming assignments from the **Operating Systems** course at [ESCOM – IPN](https://www.escom.ipn.mx/) (Escuela Superior de Cómputo, Instituto Politécnico Nacional), Bachelor's Degree in Computer Science.

The exercises cover fundamental OS concepts implemented in **C**, focusing on system-level programming, process management, and file I/O.

## 🛠️ Topics & Concepts

The following OS concepts are explored throughout this repository:

- 📁 **File Systems** – file metadata, attributes, and permissions
- 🔄 **Process Management** – creation, synchronization, and scheduling
- 🧵 **Threads** – POSIX threads, concurrency, and mutual exclusion
- 📡 **Inter-Process Communication (IPC)** – pipes, signals, shared memory
- 💾 **Memory Management** – virtual memory, paging, and segmentation
- 🔒 **Synchronization** – semaphores, mutexes, and deadlocks

## ⚙️ Getting Started

### Prerequisites

- GCC compiler
- Linux / Unix-based OS (or WSL on Windows)

### Compile & Run

```bash
# Clone the repository
git clone https://github.com/FairerFilly6/OS-ESCOM-IPN.git
cd OS-ESCOM-IPN

# Navigate to a practice folder
cd "Practica 2"

# Compile
gcc -o program main.c

# Run
./program
```
